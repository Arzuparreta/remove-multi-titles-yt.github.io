Stop A/B testing on YouTube

YouTube sometimes A/B tests different titles and thumbnails on the same video. This extension shows the original title and thumbnail from the page, so you are not stuck with whatever variant the algorithm picked.

Add to Chrome Add to Firefox

Works on youtube.com · No accounts · No network calls for titles or thumbnails

Why use it

One job, done in the page: show the title and thumbnail YouTube embedded in the document, not the headline experiment.

Consistent titles & thumbnails

See the original title and thumbnail from the page metadata instead of alternate A/B variants.

Lightweight

DOM updates only. No heavy background work and no extra requests for titles or thumbnails.

Private

No analytics or telemetry in the extension’s design—verify anytime in the open-source repo.

Get started

Install once from your browser’s store. The extension runs on YouTube automatically.

1

Install

Choose Chrome or Firefox above and confirm the store dialog.

2

Browse

3

Done
